Teague APIs - Shortened URLs

User stories

  1. I can POST a URL to https://teague-apis-short-url2.glitch.me/api/shorturl/new and I will receive a shortened URL in the JSON response.
    Example : {"original_url":"https://www.google.com","short_url":"SynhQd1YG"}
  2. If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"} HINT: to be sure that the submitted url points to a valid site you can use the function dns.lookup(host, cb) from the dns core module.
  3. When I visit the shortened URL, it will redirect me to my original link.

Example creation usage:

https://teague-apis-short-url.glitch.me/api/shorturl/new

Submit https://www.google.com

Example creation output

{ "original_url":"https://www.google.com", "short_url":"SynhQd1YG" }

Usage:

https://teague-apis-short-url.glitch.me/api/shorturl/SynhQd1YG

Will redirect to:

https://www.google.com
